Normas Técnicas em Portugal (EN 81-20)

Diagrama da Linha de Série (230V / 110V AC)

Fase / Alimentação (Manobra) [ INÍCIO ]
1. STOP Poço / Caixa (Botão de Emergência) [ OK ]
2. STOP Cabina / Tejadilho [ OK ]
3. Contacto do Limitador de Velocidade (Paraquedas) [ OK ]
4. Amortecedores / Fim de Curso Subida/Descida [ OK ]
5. Contactos das Portas de Patamar (Trincos Mecânicos) [ OK ]
6. Contacto da Porta da Cabina (Operador) [ OK ]
Bobine do Contactor Geral / Remoto Principal [ SEGURO ]

Lógica de Funcionamento

Regra de Ouro: Todos os contactos acima estão ligados eletronicamente em série. Se um único contacto falhar ou abrir, a alimentação do contactor principal é cortada instantaneamente, bloqueando o motor e aplicando o travão mecânico.
